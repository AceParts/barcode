Barcode coder : Tout pour le code barres

Ce site est à destination des développeurs.

Son objectif est de regrouper un maximum de ressources concernant les codes barres :

  • Explications détaillées
  • Générateur de code barre en ligne
  • Code source
  • ...

La librairie BCC est une librairie de génération de code barres open-source avec comme objectif d'être portée dans un maximum de langage.
Chaque portage contiendra le numéro de version de la librairie BCC ainsi que le numéro de version du portage.

Portage effectué :

  • jQuery (js)
  • Prototype (js)
  • PHP : GD / FPDF
  • Java
